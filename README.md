# UniswapPriceFeed

Program to get the live price of the Uniswap ETH / USDt pool

Output example:
```
Uniswap ETH price: '1862.4143305350908' USDt
Uniswap ETH price: '1862.390267998278' USDt
Uniswap ETH price: '1862.4770699125236' USDt
```
